100mb file 올리기
1. 기록삭제
git filter-branch --index-filter 'git rm -r --cached --ignore-unmatch yolov3_last.weights' HEAD

2. git-lfs 적용
https://git-lfs.github.com/ 이곳에 접속하여 git extension을 받아줍니다.
압축을 풀고 install.sh를 실행합니다.
레포에 가서 git lfs install 을 하여 줍니다.
git lfs track "yolo\pytorch-yolo-v3-master\cfg\yolov3_last.weights"
git add *
git commit -m "test"
git push
